JavaScript Variables

Variables
- way to save data that you can use in the future


Variable syntax:

var myName = "Nica";
where:
var - is the Keyword used to CREATE a data container - variable

= - the symbol to assign the value to a variable

myName - is the Variable name

"Nica" - is the data value


Exercise:
- print the value of a variable using alert(). 
Answer:
var myName = "Nica";
alert(myName);
